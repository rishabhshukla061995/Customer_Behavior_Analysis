# 📊 Customer Behavior Analysis – Data Analytics Project
Customer Shopping Behavior Analysis is a data-driven project exploring 3,900 purchase records to reveal insights into demographics, purchasing trends, and revenue drivers. Built with Python, SQL and Power Bi it delivers actionable strategies for boosting subscriptions, optimizing discounts, and enhancing targeted marketing.


## Overview



This project analyzes **customer purchasing behavior** using a complete data analytics workflow. The objective is to transform raw customer transaction data into meaningful insights that help understand spending patterns, product preferences, and customer segments.



The project covers the full analytics pipeline including **data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard creation, and reporting**.



By combining Python, PostgreSQL, and Power BI, the project demonstrates how data analysts convert raw datasets into **actionable business insights and visual dashboards**.



---



## Dataset



The dataset contains information about customer purchases and shopping behavior. Key attributes include:



* **Customer Information** – Customer ID, gender, age group

* **Purchase Details** – Product purchased, category, purchase amount

* **Shopping Behavior** – Discount usage, shipping type

* **Customer Engagement** – Subscription status, previous purchases

* **Product Feedback** – Review ratings



The dataset enables analysis of **customer segments, purchasing trends, and product performance**.

## Project Workflow



### 1. Data Loading



* Imported dataset using **Python (Pandas)**

* Inspected dataset structure and column types

* Checked for missing values and inconsistencies



### 2. Exploratory Data Analysis (EDA)



* Analyzed distribution of purchase amounts

* Examined customer demographics

* Identified relationships between variables

* Created visualizations to detect patterns



### 3. Data Cleaning



* Removed duplicates

* Handled missing values

* Standardized categorical variables

* Prepared dataset for analysis



### 4. SQL Analysis (PostgreSQL)



Cleaned data was loaded into **PostgreSQL** to perform analytical queries.



Examples of insights generated using SQL:



* Revenue comparison between **male and female customers**

* Customers using **discounts but spending above average**

* **Top-rated products** based on review ratings

* Spending comparison between **standard vs express shipping**

* Analysis of **subscribed vs non-subscribed customers**

* Products with **highest discount usage**

* Customer segmentation into **New, Returning, and Loyal**

* **Top products by category**

* Relationship between **repeat buyers and subscription status**

* **Revenue contribution by age group**

## How to Run the Project



### 1. Clone the Repository



```bash

git clone https://github.com/your-username/customer-behavior-analysis.git

```



### 2. Install Required Libraries



```bash

pip install pandas numpy matplotlib seaborn psycopg2

```



### 3. Run Python Analysis



* Open the **Jupyter Notebook**

* Execute the cells to perform:



  * Data loading

  * EDA

  * Data cleaning



### 4. Run SQL Queries



* Import the cleaned dataset into **PostgreSQL**

* Run the SQL queries provided in the project



### 5. View Dashboard



* Open the **Power BI (.pbix) file**

* Refresh the dataset if required
